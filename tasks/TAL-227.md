# TAL-227: Integração de Componentes do DESIGNER

## Objetivo
Integrar os componentes isolados criados pelo DESIGNER (que estão em estrutura paralela) na página principal do projeto principal Next.js (src/app/page.tsx).

## Contexto
O DESIGNER criou uma estrutura paralela (src/App.tsx) enquanto o projeto principal é Next.js. É necessário migrar esses componentes para o projeto principal, utilizando o padrão do shadcn/ui.

## Critérios de Aceitação
- [ ] Mover componentes do DESIGNER para `src/components/`.
- [ ] Integrar componentes em `src/app/page.tsx`.
- [ ] Validar estilos e funcionalidades no projeto Next.js.
- [ ] Remover a estrutura paralela (`src/App.tsx`, etc.).

DEV

## Status
cancelled

## Resolução
Cancelado em favor da migração para Tailwind CSS e shadcn/ui (concluída em TAL-260). O uso de CSS Modules não é mais necessário dado que o padrão do projeto agora é Tailwind para consistência de temas e componentes.
